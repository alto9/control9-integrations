# Hosted AI Inference

This repo does not own hosted AI inference for this initiative. If AI-assisted classification is introduced later, it must not replace deterministic policy evidence or obscure why a decision was made.

## Contract

- GitHub Actions is the first fully contracted provider; GitLab CI follows as the next expansion.
- The integration calls Control9 policy and deploy-verification APIs rather than evaluating full policy packs locally.

No hosted AI inference boundary is part of the Control9 Project Plan contracts. Deterministic evidence, explainable rules, and policy-pack versions remain the authoritative basis for decisions.
